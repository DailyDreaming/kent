  Jobs with the UCSC Genome Project
 

The Center for Genomic Sciences has two open staff positions:

  • Software Developer (#03-09-19)
  • Microarray Database and Software Developer (#03-09-20)

The primary job responsibility for the Software Developer is annotation database construction for different genomes; using various existing bioinformatics programs and shell scripts to build the base genome database and tables for numerous annotation tracks of different genomes.

Job responsibilities for both positions include:

  • Program development and implementation: programming CGI scripts in C; testing and debugging programs; working with a large Linux cluster and programming in a Linux environment.
  • Database utilization: interfacing with MySQL; adjusting software to optimize database performance.
  • Communication/Collaboration: interacting with scientific collaborators worldwide; understanding and/or learning details of the scientific goals of the Genome Project at UCSC.
  • Hardware evaluation: analyzing and evaluating hardware performance; reworking software to take advantage of hardware capabilities.
Applicants should have the following qualifications:
  • BA or BS in Computer Science or equivalent experience; significant experience working in C in a Unix or Linux environment; demonstrated experience with MySQL and managing multi-user databases with web interfacing; strong communications skills, both written and verbal; knowledge of bioinformatics methods and applications or a strong desire to learn.
  • The abilities to: study and assimilate technical structure, data and requirements of genome analysis or microarray analysis software; keep current with bioinformatics methods used at UCSC and elsewhere; quickly grasp new concepts and integrate into projects as appropriate; think on one's feet and work under pressure, with multiple deadlines; work, interact and communicate effectively with a wide range of people.
  • Preferred Qualifications: experience in molecular biology.
  • Preferred Qualifications for the Software Developer position: experience in programming languages in addition to C.
To read detailed information about the position or to submit an application, see the UCSC Employment website.
