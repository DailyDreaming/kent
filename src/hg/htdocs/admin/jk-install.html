  INSTALLING
 
  1. Check that the environment variable MACHTYPE exists on your system.  It should on Unix. (And making this on non-Unix systems is beyond the scope of this README).  For a Linux system MACHTYPE will probably be 'i686', for a Sun probably 'sparc'. If necessary set up this environment variable. 
    MACHTYPE=i686

  2. During the build, binaries are moved to your home directory: ~/bin/$MACHTYPE
    Create this directory: 
    	$ mkdir ~/bin/$MACHTYPE
  3. Set MYSQLINC and MYSQLLIBS environment variables:
    1. Set the environment variable MYSQLINC for mySQL include files. For example: 
      MYSQLINC=/usr/local/mysql/include
--or---
MYSQLINC=/usr/include
    2. Then set the MYSQLLIBS variable to point to the location of the libmysqlclient.a library and any other libraries needed on your platform to link network applications. 
      MYSQLLIBS='/usr/local/mysql/lib/mysql/libmysqlclient.a'

---Solaris for example often needs----

MYSQLLIBS='/usr/local/mysql/lib/mysql/libmysqlclient.a -lz'
  4. Create a directory named "jksrc" and go to it.

  5. Unzip jksrc.zip
    This creates the source hierarchy: ./kent

  6. Go to the kent/src directory and type 'make'.
    This builds the libraries in directories:
    • kent/src/lib
    • kent/src/jkOwnLib
    • kent/src/hg/lib


  7. Go to the kent/src/hg directory and type 'make alpha'.
    This builds the cgi binaries from directories:
    • kent/src/hg/cartReset
    • kent/src/hg/das
    • kent/src/hg/hgBlat
    • kent/src/hg/hgCoordConv
    • kent/src/hg/hgGateway
    • kent/src/hg/hgText
    • kent/src/hg/hgTracks
    • kent/src/hg/hgTracksUi
    • kent/src/hg/hgc
    • kent/src/hg/cartDump
    And the resulting binaries are moved to: /usr/local/apache/cgi-bin
    If this isn't your location for your cgi binaries, make this path a symlink to your cgi binary directory and thus the make will send the binaries to the correct location. We are working on fixing the makefiles to allow specification of a destination directory.
    Pre-compiled Red-Hat Intel x86 binaries can be found in the cgi-binaries.tgz file. Download this file and extract with: 
    	tar -xvzf cgi-binaries.tgz


  8. Many other programs are available in the source tree. For any particular program, go to its source directory and type 'make'. The resulting binary will be moved to your home directory: ~/bin/$MACHTYPE
