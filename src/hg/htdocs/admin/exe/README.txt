This directory contains applications for stand-alone use, 
separated into three directories based on which operating
system they were built for.

  - linux.i386 directory contains executables for Linux 32-bit machine.

  - linux.x86_64 contains executables for Linux 64-bit machine.

  - MacOSX directory contains executables for Mac OSX.

Download the version of the application you need that is appropriate for 
your operating system and run it from your command line.  

Run most of the applications with no arguments to see the usage statement.
