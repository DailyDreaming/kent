Table Browser

This tool allows you to download portions of the database used by the human genome browser in a simple tab-delimited text format. Please enter a position in the genome and press the submit button: (Use BLAT Search to locate a particular sequence in the genome.)

Freeze: Genome Position:

A genome position can be specified by the accession number of a sequenced human genomic clone, an mRNA or EST or STS marker, or a cytological band, a chromosomal coordinate range, or keywords from the Genbank description of an mRNA. See the User Guide for more help.
