 Labelling EST bases
 

The Genome Browser's base labelling feature allows users to quickly validate and compare EST sequences. To turn on base labelling, select the desired option from the Show sequence for ... bases pull-down menu. Spliced and Unspliced EST tracks may have base labelling turned on.

Labelling Options
all bases - Label all bases in the EST sequence.

different bases - Label only the EST bases that are different from the genome sequence.

When are the EST bases labeled?
To view the bases labelled, the option must be turned on, the track must be in squish, pack, or full display mode, and the browser must be zoomed-in to a level at which there are enough pixel per base for the text size in use. The zoom-level requirement varies with the image width of the browser. At the default image width value of 620 pixels, the annotation tracks window must be zoomed-in to at least 87 bp or fewer to view the base labelling.
