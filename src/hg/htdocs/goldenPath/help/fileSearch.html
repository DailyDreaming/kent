File Search

The Genome Browser's File Search feature allows users to find ENCODE downloadable files of interest quickly and easily.

Searching

Downloadable ENCODE files can be found by entering terms of interest in the free text Track Name and Description fields, by selecting the appropriate Group or Data Format from the drop-down menus, and/or by using the "ENCODE terms" drop-down filters.

  • ENCODE terms: This section of allows users search for files (or refine their search based on) the metadata associated with the ENCODE downloadable files. By default, there are two rows of ENCODE metadata search criteria, but more can be added by clicking the "+" button. To remove an added metadata search row, click the "-" button. Each row is comprised of two menus, which, most often, are both drop-down menus: the first containing the metadata categories available, and the second containing the the possible metadata terms available in that category. Usually the default is set to "Any" but clicking on "Any" opens up the drop-down with the possible options. Multiple options can be selected by clicking multiple check-boxes. For a small number of the categories in the first drop-down, the second menu is a free text field. Because these metadata drop-downs are assembly dependent, the options may vary from assembly to assembly. Some categories have a link to the far right of the row if there is more information available about the metadata category selected.

Results

After clicking the "Search" button, the files that meet the search criteria (up to a maxiumum of 1000 files) are displayed in a table; each row of the table represents a file meeting the critera that was set. When a search is limited to the first 1000 files, a message stating that the results were limited to 1000 is displayed.

Results Header The first column of the header contains the number of files found (up to a maximum of 1000 files) and the other colums of the header contain a header for the information about the file displayed in that column. The list of files can be sorted by clicking on the header of the column. The files can be sorted on on up to 5 attribues.

Files Click the "Download" button to download a file. Click the to go to the downloads page of the track containing that particular file (to see all the files associated with that track).
