  Contact information for the UCSC Genome Project
  Questions or problems regarding the UCSC Genome Browser software, database, genome assemblies, or release cycles: genome@soe.ucsc.edu. Messages sent to this address will be posted automatically on the genome mailing list.

Subscribe to the genome mailing list. Search the genome mailing list archives.

Server errors or other problems accessing the UCSC Genome Browser or BLAT server: genome-www@soe.ucsc.edu.

Regular weekly maintenance of the Genome Browser database occurs on Sunday mornings between the hours of 6:00 - 7:00 a.m. Pacific Time. Although we allocate an hour for this maintenance, the actual down time is usually only a few minutes.

For information on software licensing requirements, see the licensing web page. Questions about licensing the Genome Browser should be directed to: genome-www@soe.ucsc.edu.
Mirror site inquiries: genome-mirror@soe.ucsc.edu.
