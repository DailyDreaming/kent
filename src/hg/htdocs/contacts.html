Contact information for the UCSC Genome Project
 

Genome Browser contacts:
We have an active mailing list where you can post specific questions about the UCSC Genome Browser software, database, genome assemblies, and release cycle. Prior to posting a question, please review our technical support guidelines. All of the questions and answers posted to the mailing list are archived and available for searching. Many questions have been answered before; we strongly encourage you to search the archives before submitting a question to the mailing list.

  • Browse the Genome mailing list archives (indexed by thread, subject, author and date).
  • Search the Genome Browser website:  
  • Search the Genome mailing list archives:  
  • Post a question to the mailing list: genome@soe.ucsc.edu. Messages sent to this address will be posted to the moderated genome mailing list, which is also archived on a public Web-accessible pipermail archive. This archive may be indexed by non-UCSC sites such as Google.

  • Subscribe to the Genome-announce mailing list (UCSC Genome Browser project announcements only - low-volume list).
  • Subscribe to the Genome mailing list (interactive Genome Browser discussion forum - high-volume list).
Reporting server problems:
Server errors or other problems accessing the UCSC Genome Browser or BLAT server should be sent to genome-www@soe.ucsc.edu.

Regular weekly maintenance of the Genome Browser database occurs on Sunday mornings between the hours of 6:00 - 7:00 a.m. Pacific Time. Although we allocate an hour for this maintenance, the actual down time is usually only a few minutes. Occasionally, additional maintenance is scheduled by the School of Engineering. These service interruptions are posted on the Genome Browser home page.

Commerical licensing:
Questions about commercial licensing of the Genome Browser should be sent to genome-www@soe.ucsc.edu. For more information about licensing, see the Genome Browser software licensing requirements.

Questions regarding commercial licensing of Blat should be directed to jim_kent@pacbell.net. See the Genome BLAT web site for more information.

Mirroring:
The genome-mirror mailing list is a forum to which licensed and non-commercial sites can post questions about the setup and maintenance of Genome Browser mirrors. Note that commercial use of the source code requires a license.
  • Post a mirror site inquiry: genome-mirror@soe.ucsc.edu. Messages sent to this address will be posted to the moderated genome-mirror mailing list, which is archived on a public Web-accessible pipermail archive. This archive may be indexed by non-UCSC sites such as Google.
  • Subscribe to the Genome-mirror mailing list (UCSC Genome Browser mirror site questions and announcements only - low-volume list).
  • Display a list of mirror sites.
  • Read procedures for creating a mirror site.
