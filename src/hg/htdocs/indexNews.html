4 February 2009 - New Functionality: in-silico PCR Tool

Many Genome Browser users are familiar with UCSC's in-silico PCR tool, which searches a sequence database with a pair of PCR primers, using an indexing strategy for fast performance. In response to requests from users and staff, we have extended the functionality of this tool to allow the query of transcribed sequences. Using primers within the exonic regions of a gene, you can see a graphical view of the predicted PCR product. The gene is shown in all of its isoforms with the introns removed.

This new functionality, which is based on the UCSC Genes annotation, is currently available for human assemblies hg16, hg17, and hg18, and mouse assemblies mm8 and mm9. To try it out, select one of the supported assemblies from the In-Silico PCR Genome and Assembly menus, and then choose the "UCSC Genes" option from the Target menu.
