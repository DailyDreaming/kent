7 Mar. 2008 - New Reverse Functionality Released

The Genome Browser is happy to announce a new functionality in the main genome track display. A new button, "Reverse," is available which allows users to view the entire browser image flipped right-to-left. This is especially useful when a user's gene of interest aligns on the opposite strand from reference assembly and shows those genes with the in the 5' to 3' direction.

When the reverse function has been activated, all the of track labels usually displayed on the left side of the track (including the minibutton which provides access to configuration options) are displayed on the right side. This allows users to grok at a glance which direction is being displayed.

Thanks to Mark Diekhans, Ann Zweig, Robert Kuhn and the rest of the engineering team.
