27 May 2011 - New Release of UCSC Genes for Mouse

We've released an updated set of UCSC Genes for the mm9 (NCBI Build 37) mouse Genome Browser. This version of the gene set was generated using the same computational pipeline as the previous mm9 UCSC Genes, but is based on more recent GenBank data.

The new release has 55,419 total transcripts, compared with 49,409 in the previous version. The total number of canonical genes has increased from 27,389 to 28,661.

Comparing the new gene set with the previous version:

  • 33,977 transcripts did not change between versions
  • 69 transcripts were not carried forward to the new version
  • 13,701 transcripts are compatible with those in the previous set
  • 1,662 transcripts overlap with those in the previous set
