About the ENCODE Experiment Matrix

The ENCODE Experiment Matrix is a set of web pages that visually summarize the types of data produced by the ENCODE project. This page shows the number of experiments by cell type and assay. The companion Experiment Summary page shows the number of experiments by assay type alone, and includes annotations that are cell-type independent (annotations on the reference genome). The ChIP-seq Experiment Matrix page provides a more detailed view of the chromatin immunoprecipitation experiments, showing experiments by cell type and antibody target.

An ENCODE experiment is defined as a biochemical assay and follow-on data analyses performed on a single cell type by a single lab. An experiment will usually have multiple browser tracks to display differing views of the data (e.g. enrichment signal graph, peak calls), and multiple files to support analysis (e.g. sequence alignments in BAM format, signal graph in bigWig format), for multiple replicates. Each ENCODE experiment has a unique accession assigned by the ENCODE Data Coordination Center (e.g. wgEncodeH00001).

To display or download ENCODE data from this page, select tracks or files in the panel. Then click the desired box in the matrix. This will open a search window with the Track Search or File search tool showing the full set of browser tracks or downloadable files for the experiments selected so you can configure viewing in the browser or refine the list of download files.

These pages are best viewed in Firefox, Chrome, or Safari browsers.
