Description & Credits

Twinscan predicts genes in a manner similar to Genscan, except that Twinscan takes advantage of genome comparison to improve gene prediction accuracy. More information and a web server can be found at http://genes.cs.wustl.edu/ .

The Twinscan algorithm is described in:

Korf, I., P. Flicek, D. Duan, and M.R. Brent. 2001. Integrating genomic homology into gene structure prediction. Bioinformatics 17, S140-148.

This paper can downloaded from http://bioinformatics.oupjournals.org/cgi/reprint/17/suppl_1/S140.pdf
