Description

This track displays the level of conservation for each of 12 zoo species, based on blastz pairwise alignments.

Methods

Blastz pairwise alignments were performed between the NISC-generated sequence for each species and the UCSC compiled human reference sequence. All sequences were RepeatMasked, where repetitive sequences were denoted by lower-case letters. The blastz program will not initiate alignments within repetative regions, but it will extend alignments into repetitive regions.

Gray horizontal bars denote a mapping/sequencing gap and percent identity less than 50% is not shown since it falls below the minimum y-axis value.
