Description

This track displays alignments of the NIA Mouse Gene Index (Version 3) against the mouse genome. The index was assembled from the BLAT alignments of RefSeq (N = 23,345; 10/27/2003), Ensembl (N = 34,076; 02/13/2004), Riken (N = 60,484; 10/27/2003), GenBank (N=42,040), dbEST (N = 4,088,827; 01/22/2004), and NIA (N=370,460) to the UCSC Mouse Genome (October 2003; mm4). See http://lgsun.grc.nia.nih.gov/geneindex3 for more information.
