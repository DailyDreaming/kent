Description

This track is BLASTP run for proteins in S. solfataricus against various categories. It was built using the NCBI Link version of blastp along with the NCBI protein database of GIs from Archaea, Bacteria and Eukarya. The protein information for each gene was also taken from NCBI. The cutoff was set with an e value of 10^-3. The alignments are 60 pluses("+") long with the first one being the complete protein of S. solfataricus.
