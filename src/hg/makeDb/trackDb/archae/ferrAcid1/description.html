Sample position queries

A genome position can be specified chromosomal coordinate range, COG ID, or keywords from the GenBank or TIGR description of an gene. The following list shows examples of valid position queries. 

Request:
    Genome Browser Response:

chr
Displays all of chromosome (the entire genome).
chr:1-10000
Displays first ten thousand bases on the chromosome.
NADH
Lists genes with "NADH" in the description.
FA0002
Display chromosome at position of gene FA0002.
FA000
Lists genes with names beginning with "FA000".
