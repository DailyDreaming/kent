The Ferroplasma acidarmanus genome sequence is approximately 1,865,440 bp long and contains about 1695 protein coding genes.  F. acidarmanus is an acidophilic iron-oxidizing archaeon that is native to metal-rich environments (including those rich in arsenic). It grows optimally at 42C and pH 1.2.

The genome is currently being sequenced by the DOE Joint Genome Institute.

Sample position queries

A genome position can be specified chromosomal coordinate range, COG ID, or keywords from the GenBank or TIGR description of a gene. The following list shows examples of valid position queries. 

Request:
    Genome Browser Response:

chr
Displays all of chromosome (the entire genome).
chr:1-10000
Displays first ten thousand bases on the chromosome.
NADH
Lists genes with "NADH" in the description.
FA0002
Display chromosome at position of gene FA0002.
