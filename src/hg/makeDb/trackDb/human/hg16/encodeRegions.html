Description

This track depicts target regions for the NHGRI ENCODE project, which is described in The ENCODE Project: ENCyclopedia of DNA Elements.

There are 44 ENCODE regions, comprising 1% of the human genome. These regions were selected for intensive study to identify, locate and analyze all functional elements in the region. For a description of how the regions were selected, see The ENCODE Project: Target Selection Process.

For a frame view of the UCSC browser with a menu containing the ENCODE regions, use The ENCODE Project: Target Selection Process.
