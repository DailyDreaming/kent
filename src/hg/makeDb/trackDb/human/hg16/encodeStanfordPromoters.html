Description

This track displays activity levels of predicted promoters in the ENCODE regions, based on luciferase reporter assays. Activity level is indicated by color, ranging from black (no activity) to red. Experiments were performed on 14 cell lines:

Methods

Promoters in the ENCODE region were predicted using a variation on methods previously described (Trinklein et al., 2003, Trinklein et al., 2004). Any human cDNAs in the database with at least one base pair of exon overlap were merged. This generated the gene models. The transcription start sites were predicted by starting with the 5' end of each gene model as well as other 5' ends that were at least 500 base pairs away from the 5' end. Promoters were identified as the approximately 600 base pairs upstream and 100 base pairs downstream of each transcription start site.

Primer3 was used to pick primers yielding approximately 500 base pair fragments containing the transcription start site. Each fragment of DNA represented in this track was cloned into a luciferase reporter vector (pGL3-Basic, Promega) using the BD Clontech Infusion Cloning System. The Dual Luciferase system (Promega) was used to co-transfect the experimental DNA along with a control plasmid expressing Renilla in 96-well format into one of the fourteen cell types using FuGENE Transfection Reagent (Roche). Each transfection was done in duplicate.

Values here represent averages of the Luciferase-to-Renilla ratio. To determine fold increase, 96 random genomic fragments were cloned and tested, using the average values to determine a threshold (3 standard deviations) for functional promoters.

Any promoters that did not meet the threshold for function have a value of zero; those that are positive have been scaled to show graded color on the browser.

Verification

Data were verified by repeating the preparation and measurement of 48 random fragments. No significant variation between the two preparations was detected.
