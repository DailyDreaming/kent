Description

This track displays activity levels of predicted promoters in the ENCODE regions, based on Luciferase reporter assays. Activity level is indicated by color, ranging from black (no activity) to red. Experiments were performed on 14 cell lines:

Methods

Using a variation on methods previously described we predicted all promoters in the ENCODE region (Trinklein, et al. 2003, Trinklein et al. 2004). We took all human cDNAs in the database and merged any cDNAs with at least one base pair of exon overlap. This generated our gene models. To predict transcription start sites, we started with the 5-most end of each gene model as well as other 5 ends that were at least 500 base pairs away from the 5-most end. Promoters were identified as the approximately 600 base pairs upstream and 100 base pairs downstream of each transcription start site.

Primer3 was used to pick primers yielding approximately 500 base pair fragments containing the transcription start site. Each fragment of DNA represented in this track was cloned into a luciferase reporter vector (pGL3-Basic, Promega) using the BD Clonetech Infusion Cloning System. We used the Dual Luciferase system (Promega) to transfect the experimental DNA with a control plasmid expressing Renilla in 96-well format into one of the fourteen cell types using Fugene Transfection Reagent (Roche). Each transfection was done in duplicate.

Values here represent averages of the Luciferase to Renilla ratio. To determine fold increase, we have also cloned and tested 96 random fragments in the genome. We used the average values of these 96 to determine a threshold (3 standard deviations) for functional promoters.

Any promoters that did not meet the threshold for function have a value of zero and those that are positive have been scaled to show graded color on the browser.

Verification

Data was verified by repeating the preparation and measurement of 48 random fragments. We saw no significant variation between the original preparation and the second preparation of these fragments.
