Description

DNaseI-hypersensitive sites were cloned from primary human CD4+ T-cells and sequenced using massively parallel signature sequencing (MPSS; Brenner et al., 2000; Crawford and Collins, unpublished data). Only clusters of multiple DNaseI library sequences that map within 500 bases of each other are displayed. Each cluster has a unique identifier; the last digit of each identifier represents the number of sequences that map within that particular cluster.

Verification

Real-time PCR assay was used to verify that 90% of the clusters identify valid DNaseI-hypersensitive sites .

This data set includes confirmed elements for 35 of the 44 ENCODE regions. It is estimated that this data identifies 10-20% of all HS sites within CD4+ T cells. Further sequencing will be required to identify additional sites.
