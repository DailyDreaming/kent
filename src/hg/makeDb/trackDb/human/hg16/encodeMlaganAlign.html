Description

This track displays a set of human-centric multiple alignments of the sequencies included in the May 2005 ENCODE freeze, generated by MLAGAN.

Methods

The alignments were created in the following way:

  • First, the sequence of each other species is "rearranged" so that it is orthologously collinear with respect to the human sequence. This is done with a suite of tools and algorithms, based on Shuffle-LAGAN and SuperMap. For every pair of <human, other species>, Shuffle-LAGAN is used to find the best-scoring chain of local similarities (according to some scoring scheme that penalizes evolutionary rearrangements), and then SuperMap is used to aggregate parts of the chain into a human-monotonic map of syntenic blocks. This mapping is used to undo the genomic rearrangements of the other sequence and convert it to a form that is directly alignable to the human sequence.
  • Then, a multiple global alignment is created for every region using MLAGAN.
  • Finally, the alignments are refined using MUSCLE. MUSCLE processes small non-overlapping windows of the alignment and attempts to realign them in an iterative fashion, keeping the refined alignment if it has a better sum-of-pairs score than the original.
