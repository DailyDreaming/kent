Description

This track displays a set of human-centric multiple alignments of the sequencies included in the May 2005 ENCODE freeze, generated by MLAGAN.

Methods

The alignments were created in the following way: First, the sequence of each other species is "rearranged" so that it is orthologously collinear with respect to the human sequence. This is done with a modified version of SuperMap, based on Shuffle-LAGAN. Then, a multiple global alignment is created for every region using MLAGAN. Finally, the alignments are refined using MUSCLE, which realigns some parts in a limited-size window, in an iterative fashion.
