Description

This track displays the predicted hydroxyl radical cleavage intensity for each nucleotide in the ENCODE regions. As the hydroxyl radical cleavage intensity is proportional to the solvent accessible surface area of the deoxyribose hydrogen atoms (Balasubramanian et al., 1998), this track represents a structural profile of the DNA in the ENCODE regions.

Please visit the Tullius lab ENCODE data website if you wish to download the datasets. Also, check out our visualization tool to make plots using any smoothing window you prefer.

Methods

Hydroxyl radical cleavage intensity predictions were performed using an in-house sliding trimer window (STW) algorithm. This algorithm draws data from our ·OH Radical Cleavage Intensity Database (ORChID), which contains >150 experimentally determined cleavage patterns. These predictions are fairly accurate, with a Pearson coefficient of ~0.85 between the predicted and experimentally determined cleavage intensities. For more details on the hydroxyl radical cleavage method, please see the references below.

Verification

The STW algorithm has been cross-validated by removing each test sequence from the training set and performing a prediction. The mean correlation coefficient (between predicted and experimental cleavage patterns) from this study was 0.85.
