Description

This track is the union of the eleven subtracks from the Intronic Proximal composite track collapsed into this single track.

This track is the collection of all the TransFrags/TARS from the Yale and Affy transfrags tracks, selected to be within the transcript coding region of the Gencode genes, and within 5000 bases of the exon coding regions of the Gencode genes. Thus, this set are the TransFrags/TARS within the introns of the Gencode genes and nearby to the exons.

Methods

This data was collected directly from the UCSC Genome browser via the Table Browser.
