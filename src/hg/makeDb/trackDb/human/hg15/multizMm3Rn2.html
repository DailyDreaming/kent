Description

This track displays multiz alignments of the Feb. 2003 mouse draft assembly and the rat Jan. 2003 assembly to the human genome.

Methods

Multiz is a multiple alignment program that takes blastz/axtBest alignments as input. It uses the same scoring matrix as blastz between pairs of sequences: The overall score is the sum of the score between all pairs.

Credits

Multiz was created by Webb Miller from the Penn State Bioinformatics Group. Jim Kent wrote scripts to run multiz genome-wide and to display the alignments in this browser. Mouse sequence data is provided by the Mouse Genome Sequencing Consortium. Rat sequence data is provided by the Rat Sequencing Consortium.
