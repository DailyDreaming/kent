Description

This track shows the cytogenetic locations of all enries in the Online Mendelian Inheritance in Man (OMIM) database.

Display Conventions and Configuration

Cytogenetic locations of OMIM entries are displayed as solid blocks. The entries are colored according to the OMIM phenotype class of associated disorders:

  • Lighter Green for Class 1 OMIM records - the disorder is placed on the map based on its association with a gene, but the underlying defect is not known.
  • Light Green for Class 2 OMIM records - the disorder has been placed on the map by linkage; no mutation has been found.
  • Dark Green for Class 3 OMIM records - the molecular basis for the disorder is known; a mutation has been found in the gene.
  • Purple for Class 4 OMIM records - a contiguous gene deletion or duplication syndrome, multiple genes are deleted or duplicated causing the phenotype.
  • Light Gray for Others - no associated OMIM phenotype class info available.

Gene symbol and disease information, when available, are displayed on the details page for an item.

On the main browser display, the description of the OMIM entry are shown when Full display mode is chosen. In Pack mode, the entry description is shown when mousing over each entry.

Methods

This track was constructed as follows:

  • The OMIM genemap data file was downloaded from ftp://ftp.ncbi.nih.gov/repository/OMIM/genemap. The data are loaded into an MySQL table, omimGeneMap.
  • The cytogenetic location data (from the location column in omimGeneMap table) are parsed and converted into genomic start and end positions based on the cytoBand table. These genomic positions, together with the corresponding OMIM IDs, are saved into a BED file and loaded into table omimLocation. All OMIM gene entries as reported in the "OMIM Genes" track are excluded from this table.
