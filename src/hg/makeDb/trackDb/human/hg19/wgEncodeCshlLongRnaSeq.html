Description

These tracks are high throughput RNA sequencingi from ENCODE cell lines generated by the ENCODE consortia. They contain information about human RNAs > 200 nucleotides obtained as short reads off the Illumina GAIIx platform. Data is available from biological replicates of several cell lines. In addition to profiling Poly-A+ RNA samples from whole cells there is also data from various subcellular compartments as well and profiles from Poly-A depleted (-) RNA. In many cases, there are also CAGE, Small RNA-Seq and PET-diTAG datasets available from the same biological replicates.

Display Conventions and Configuration

This track is a multi-view composite track that contains the following views:
Alignments
The Alignments view shows reads mapped to the genome. Sequences determined to be transcribed on the positive strand are shown in blue. Sequences determined to be transcribed on the negative strand are shown in red. Sequences for which the direction of transcription was not able to be determined are shown in black. Please see the SAM Format Specification for more information on the SAM/BAM file format.
Raw Signals
Plus Raw Signal (bigWig) shows the density of aligned tags that map to the genome between 1-10 times on the plus strand.
Minus Raw Signal (bigWig) shows the density of aligned tags that map to the genome between 1-10 times on the minus strand.
All Raw Signal (bigWig) shows the density of aligned tags that map to the genome between 1-10 times on both strands.

Methods

Cell culture Cells were grown according to the approved ENCODE cell culture protocols.

Biological Replicates Whenever possible we grow up and assay RNA from 2 biological replicates. Each replicate is issued a unique ID# (e.g. 001). If RNA from multiple cellular compartments is desired, the batch will be partitioned just prior to RNA isolation and the ID# appended with its compartment label (e.g. 001WC = Whole Cell RNA from replicate 001). These bio-replicate ID's are constant across experiments allowing the improved data comparison. (WC) Whole Cell RNA, (N) Nuclear RNA, (C) Cytoplasmic RNA, (NP) Nucleoplasm, (NL) Nucleolus, (CH) Chromatin, (+) Poly-A+ RNA, (-) Poly-A- RNA.

Exogenous RNA spike-ins, still in development at NIST were added to each bioreplicate and carried through library construction. Library insert sizes are ~ 200 bases. Accompanying each RNA-Seq dataset we submit is a Production Document. This document contains details about the RNA isolations and treatments, library construction, spike-ins as well as quality control figures relevant for that library.

Sequencing and Mapping The data are generated on the Illumina GAIIx platform in pair-end 76 base format to a current depth of ~200 million reads. The data are mapped using Spliced Transcript Alignment and Reconstruction (STAR) written by Alex Dobin, CSHL (in preparation). STAR is able to map split reads in an annotation agnostic fashion allowing for identification of novel elements like splice junctions and also takes into account the quality of the sequence itself trimming of mismatched bases at the read ends. ???? Verification

We take several steps to ensure that the data being generated is high quality and reproducible. Each lane is mapped independently and FPKMs are computed for Gencode.v3c protein-coding transcripts. Lanes of data from the same library, i.e. technical replicates are only merged together if their FPKMs have a Spearman R value > 0.98 and show similar mapping stats. We check to see that we recover a broad range of spike-ins in their indicated concentrations. Additionally, we assess the strandedness of the library by verifying that >98.5% of the reads that map to spike-ins map to the sense strand. To assess biological reproducibility we only submit biological replicates where the FPKMs of Genocde transcripts have a Spearman R value >0.95.

Release Notes

This is Release 1 (Jan 2011) of this track for hg19. It includes data from the hg18 releases that has been remapped to hg19 (GM12878 and K562 Total RNA whole cell and K562 PolyA+ RNA cytosol) as well as novel data from 13 additional cell lines and 4 additional cell localizations, as well as some PolyA- samples. The singnals are now in bigWig format. The alignments are in BAM format. The newer data all have Bio Replicate numbers and Protocol ID numbers that can be used to link the samples with other experiments.
