Description

This track consolidates variants from locus specific databases into a single track. It currently has entries from the HbVar database.

Variant Sources

  • The HbVar database has variants found in the globin loci. The database can be found at http://www.bx.psu.edu/.
  • Coming soon, CFTR data. Currently a few sample variants from this region have been added.
  • Possible connection with WayStation. HGVS community-wide database. http://www.genomic.unimelb.edu.au/mdi/centraldb.html

Filtering

  • Filtering can be done on the location of the mutation. This is the location with respect to genes, eg. exclude exons, introns, 5' utr, 3' utr or not within known transcription unit.
  • Filtering can also be done by excluding types of the mutations. The types are substitution, insertion, deletion, duplication, and complex.

Supporting Details

Positional information can be found in the annotations section of the Genome Browser downloads page, which is organized by species and assembly. Non-positional information can be found in the shared data section of the same page.
