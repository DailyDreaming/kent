Description

Hits for the Sanger PCR array ChIP-chip platform

Methods

Hits are found using an HMM analysis which produces both "hit regions" and the peak centers. The submitted tracks contain the ChIP-chip data, the hit regions, and the hit centers. We recommend that others wanting the hits take the peak centers and expand them to whatever size your analysis uses.
