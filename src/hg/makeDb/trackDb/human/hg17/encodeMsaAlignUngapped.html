Description

This track displays the number of ungapped species for every human base in the multiple-sequence alignments in ENCODE regions in the Sep 2005 ENCODE MSA freeze, which were generated by the programs MAVID, MLAGAN and TBA (v2).

Methods

The number of ungapped species for every human base in every alignment is displayed. This track complements the Alignment Agreement track (Align Agree), and is useful for distinguishing between agreement on gaps versus agreement on matched bases.
