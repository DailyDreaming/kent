Description

Consolidates regulatory regions and other useful landmarks from locus-specific databases into a single track. This prototype version includes entries mostly from the beta globin locus, plus a few entries from other regions.

Data Filtering

  • Filtering can be performed on the type of the region. Types include regulatory, promoter, enhancer, and hypersensitive site.
