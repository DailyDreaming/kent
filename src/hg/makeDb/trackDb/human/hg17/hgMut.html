Description

Consolidates mutations from locus-specific databases and a genome wide mutation database into a single track. This prototype version includes entries mostly from the globin loci, plus a few sample entries from the CFTR region for the locus-specific sources.

Display Conventions and Configuration

Color key: The track is color coded by the mutation type.

  • substitution = purple
  • insertion = green
  • deletion = blue
  • duplication = orange
  • complex = brown
  • unknown = black

Labels: By default items are labeled by the HGVS name. To change this click the desired option or options. None will produce no labels no matter what else is selected.

Variant Sources

  • The HbVar database describes variants and thalassemia mutations in the globin loci. http://www.bx.psu.edu/
  • The PAHdb has mutations in the phenylalanine hydroxylase (PAH) gene. http://www.pahdb.mcgill.ca/
  • Coming soon: CFTR data. (A few sample variants from this region have already been added.)
  • Swiss-Prot data on variants within coding regions is available genome wide.
  • A possible connection with WayStation, the HGVS community-wide database. http://www.centralmutations.org/

Data Filtering

  • Filtering can be performed on the location of the mutations with respect to genes, i.e. to exclude those in exons, introns, UTR regions, or those not within any known transcription unit.
  • One can also exclude mutations by type: substitution, insertion, deletion, duplication, or complex.
  • Filtering by source is also provided. The locus specific databases are listed as one source.
