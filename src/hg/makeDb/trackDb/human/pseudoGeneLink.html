Description

The set of retroposed copies of genes that have been inserted since the mouse/human split. The set of insertions are either functional genes that have acquired a promoter from a neighboring gene or non-functional pseudogenes.

Methods

This track was generated as follows:

  • Align all mRnas (build 139) to genome using Blastz assembly to the human genome.

  • Find alignments with and without introns
  • Extract biological features
  • PolyA tail inserted in genome
  • Remove introns > 80% SINE/LINE
  • Check position of splicing in pseudogene
  • Break in synteny with respect to mouse
  • Target site duplications
  • Other alignment features
