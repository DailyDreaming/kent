Description

This track shows the genomic positions of all gene enries in the Online Mendelian Inheritance in Man (OMIM) database.

Display Conventions and Configuration

Genomic locations of OMIM gene entries are displayed as solid blocks. The entries are colored according to the OMIM phenotype class of associated disorders:

  • Lighter Green for Class 1 OMIM records - the disorder is placed on the map based on its association with a gene, but the underlying defect is not known.
  • Light Green for Class 2 OMIM records - the disorder has been placed on the map by linkage; no mutation has been found.
  • Dark Green for Class 3 OMIM records - the molecular basis for the disorder is known; a mutation has been found in the gene.
  • Purple for Class 4 OMIM records - a contiguous gene deletion or duplication syndrome, multiple genes are deleted or duplicated causing the phenotype.
  • Light Gray for Others - no associated OMIM phenotype class info available.

Gene symbol and disease information, when available, are displayed on the details page for an item.

On the main browser display, the description of the OMIM entry are shown when Full display mode is chosen. In Pack mode, the entry description is shown when mousing over each entry.

Methods

The mappings displayed in this track are based on the mapping between OMIM gene entries and ENTREZ gene IDs:

  • The data file, genemap.txt, from OMIM is loaded into an MySQL table, omimGeneMap.
  • The data file, mim2gene.txt, from OMIM is loaded into an MySQL table, mim2gene.
  • Entries in genemap.txt having disorder info are parsed and loaded into the omimPhenotyp table.
  • From the mim2gene table, for each OMIM gene, its ENTREZ gene ID is used to get the corresponding RefSeq gene ID via the refLink table. The RefSeq gene genomic chrom, start, and end data are obtained from the refGene table. Together with the OMIM gene ID, a bed 4 table omimGene2 is constructed as the primary table for this track.
  • Links to related RefSeq Genes and UCSC Genes are listed on the details page (using Entrez gene ID as cross-reference and other info from mim2gene, refLink, and kgXref tables).
