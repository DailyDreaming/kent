Description

These tracks display the level of sequence uniquness of the reference hg18 genome. They were generated using different window sizes and high signal will be found in areas where the sequence is not unique.

Methods

The Duke uniqueness tracks were generated by self-alignment of a sliding window of fixed size across the entire genome. The signal is essentially the count of alignments of one window to the rest of the genome, normalized to between 0 and 1, with 0 representing unique sequence. The window is shifted by 1bp so that each base has a signal that represents the degree to which the sequence surrounding that base is unique. Thus, if the window size is 20bp then a 0 signal at a single base means that the sequence of 39 bases which has that one base at its center is unique within the entire genome. A signal of 1 means that there are at least ???40??? regions within the genome which align to the region containing the one base.
