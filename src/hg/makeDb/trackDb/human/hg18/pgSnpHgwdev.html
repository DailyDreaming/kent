Description

This track displays variant base calls from personal genomes that have been made publicly available.

Display Conventions and Configuration

The alleles in the the details page are what is found on the plus strand. The nucleotide bases shown in the browser adjusts to the requested view, complementing and or reversing the nucleotides as needed.

The frequency shown by box color, mouse overs, and on the details page is the read frequency of the alleles.

Methods

Yoruban NA18507 (SOLiD) (Applied Biosystems)
Applied Biosystems released the read sequences (color-space FASTA) to the NCBI Short Read Archive. Aakrosh Ratan in the Miller Lab at Pennsylvania State University (PSU) mapped the sequence reads to the reference genome and called single-base variants using SOLiD&tm; System Analysis Pipeline Tool (Corona Lite).

CEU trio NA12878, NA12891, NA12892; YRI daughter NA19240 (1000 Genomes)
