Description

There are two subtracks on the Poly(A) composite tracks. One is the PolyA_DB subtrack, which are reported poly(A) sites. The other is the Poly(A) prediction subtrack.

The PolyA_DB database is a set of $organism mRNA polyadenlyation sites based on EST/cDNA evidence. The track has 28,857 annotated poly(A) cleavage sites. This is a single base denoting the beginning of a poly(A) tail in a nascent mRNA transcript and is typically 10-30 nucleotides downstream of a PAS (polyadenylation signal), the most common being AAUAAA (or AATAAA on the DNA). The PolyA_DB web server is found at http://polya.umdnj.edu/polyadb.

The Poly(A) predictions are made using 1500 base DNA sequences centered at the end of each RefSeq gene. The sequences are then inputted into the support vector machine described in Cheng, et. al. The SVM scores each base based on a model using 15 different cis elements. High-scoring regions are highlighted with the highest-scoring base getting a thicker line. The median length of these regions is 48 bases.
