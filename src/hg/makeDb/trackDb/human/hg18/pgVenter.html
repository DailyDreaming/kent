Description

This track contains Venter's SNPs. An overview is given here.

Display Conventions and Configuration

The alleles in the the details page are what is found on the plus strand. What is shown in the browser adjusts to the requested view, complementing and or reversing the nucleotides as needed.

Methods

This track contains Venter's SNPs from the file HuRef.InternalHuRef-NCBI.gff. This was filtered for method 1 SNPs only, and any SNPs that had N as an allele were also filtered out.

Credits

Sequence data was obtained from The J. Craig Venter Institute website at www.jcvi.org.
