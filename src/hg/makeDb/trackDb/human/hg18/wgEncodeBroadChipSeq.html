Description

This super-track displays related tracks of chromatin state maps generated by the Broad/MGH ENCODE group using ChIP-Seq.

The ChIP-Seq method involves crosslinking histones and other DNA associated proteins to genomic DNA within cells using formaldehyde. The crosslinked chromatin is subsequently extracted, mechanically sheared, and immunoprecipitated using specific antibodies. After reversal of crosslinks, the immunoprecipitated DNA is sequenced and mapped to the human reference genome. The relative enrichment of each antibody-target (epitope) across the genome is inferred from the density of mapped fragments.

The following antibodies have been used to map chromatin state in each celltype:

For each antibody and celltype, two subtracks are provided:

Fragment density: These subtracks represent the number of sequenced and uniquely aligned ChIP fragments across the human genome at 25bp resolution. A value of -1 indicates a masked (non-unique) position.

Enriched intervals: These subtracks represent discrete intervals predicted to be significantly enriched with ChIP fragments using a statistical model. Each interval is associated with an enrichment ratio and a p-value.

This super-track also contains a single special track called Alignable, which marks unmasked (unique) positions in the reference genome as 1s and masked (non-unique) positions as 0.

Methods

Growth protocol

Tier 1 (K562, GM12878) and Tier 2 (HepG2, HeLa-S3, HUVEC, Keratinocyte) cells were cultured according to shared ENCODE protocols.

Extraction protocol

Cells were fixed in 1% formaldehyde and resuspended in lysis buffer. Chromatin was sheared to 200-700bp using a Diagenode Bioruptor. Solubilized chromatin was immunoprecipitated with antibodies against H3K4me3 (Abcam 8580), H3K4me2 (Abcam 7766),

H3K4me1 (Abcam 8895), H3K9ac (Abcam 8896), H3K27ac (Abcam 4729), H3K27me3 (Millipore 07-449), H3K36me3 (Abcam 9050), H3K9me1 (Abcam 8896),

H4K20me1 (Abcam 9051) or Pol2(b) (Covance MMS-128P). Antibody-chromatin complexes were pulled-down using protein A-sepharose (or anti-IgM-conjugated agarose for RNA polymerase II), washed and then eluted. After cross-link reversal and proteinase K treatment, immunoprecipitated DNA was extracted with phenol-chloroform, ethanol precipitated, treated with RNAse and purified. One to ten nanograms of DNA were end-repaired, adapter-ligated and sequenced by Illumina Genome Analyzers as recommended by the manufacturer.

Data processing

Sequence reads from each IP experiment were aligned to the human reference genome (hg18) using the ARACHNE computational pipeline. First, a table was pre-computed to associated all possible 12-mers with all of their occurrences in the genome. For each ChIP-Seq read (forward and revere complement orientation), each potential start point was then found and the number of mismatches in the corresponding gap-free alignment was computed. All uniquely aligned reads (defined as the second to best alignment having >2 mismatches more than the best alignment, and the total mismatch count being <=6) were kept. Fragment densities were computed by counting the number of reads (extended to 300 bp) overlapping each position in the genome (at 25bp resolution). Non-unique positions in the reference genome were pre-computed by aligning every 36-mer in the genome to the whole genome and masking positions that did not meet the uniqueness criteria defined above. Discrete intervals of ChIP-Seq fragment enrichment were identified using a scan statistics approach, under the assumption of uniform background signal across the unique portion of the genome.
