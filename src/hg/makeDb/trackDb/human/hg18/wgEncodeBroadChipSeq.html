Description

This super-track displays related tracks of chromatin state maps generated by the Broad/MGH ENCODE group using ChIP-Seq.

The ChIP-Seq method involves crosslinking histones and other DNA associated proteins to genomic DNA within cells using formaldehyde. The crosslinked chromatin is subsequently extracted, mechanically sheared, and immunoprecipitated using specific antibodies. After reversal of crosslinks, the immunoprecipitated DNA is sequenced and mapped to the human reference genome. The relative enrichment of each antibody-target (epitope) across the genome is inferred from the density of mapped fragments.

For each antibody and celltype, two subtracks are provided:

Signal: These subtracks represent the number of sequenced and uniquely aligned ChIP fragments across the human genome at 25bp resolution.

Peaks: These subtracks represent discrete intervals predicted to be significantly enriched with ChIP fragments using a statistical model. Each interval is associated with an enrichment ratio and a p-value.

This group of tracks also contains a single special track called Alignability, which marks unique positions in the reference genome as 1s and non-unique positions as 0.

Methods

Growth protocol

Tier 1 (K562, GM12878) and Tier 2 (HepG2, HeLa-S3, HUVEC, Keratinocyte) cells were cultured according to shared ENCODE protocols.

Extraction protocol

Cells were fixed in 1% formaldehyde and resuspended in lysis buffer. Chromatin was sheared to 200-700bp using a Diagenode Bioruptor. Solubilized chromatin was immunoprecipitated with antibodies against H3K4me3 (Abcam 8580), H3K4me2 (Abcam 7766),

H3K4me1 (Abcam 8895), H3K9ac (Abcam 8896), H3K27ac (Abcam 4729), H3K27me3 (Millipore 07-449), H3K36me3 (Abcam 9050), H3K9me1 (Abcam 8896),

H4K20me1 (Abcam 9051) or Pol2(b) (Covance MMS-128P). Antibody-chromatin complexes were pulled-down using protein A-sepharose (or anti-IgM-conjugated agarose for RNA polymerase II), washed and then eluted. After cross-link reversal and proteinase K treatment, immunoprecipitated DNA was extracted with phenol-chloroform, ethanol precipitated, treated with RNAse and purified. One to ten nanograms of DNA were end-repaired, adapter-ligated and sequenced by Illumina Genome Analyzers as recommended by the manufacturer.

Data processing

Sequence reads from each IP experiment were aligned to the human reference genome (hg18) using MAQ with default parameters, except C=10 (maximum number of alignments/read <= 10). Fragment densities were computed by counting the number of reads overlapping each position in the genome (counted as 1 from the start of each alignment to 200 bp downstream and by 0.25 from 200 to 300 bp, and displayed at 25bp resolution). Discrete intervals of ChIP-Seq fragment enrichment were identified using a scan statistics approach, under the assumption of uniform background signal.
