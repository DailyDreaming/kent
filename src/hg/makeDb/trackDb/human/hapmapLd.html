Description

This track shows linkage disequilibrium between pairs of SNPs as genotyped by the HapMap consortium.

In full mode, each diagonal represents a different SNP, with each square representing a pairwise comparison between two SNPs. By convention, shades of red are used to indicate statistically significant (LOD >= 2) allelic association (linkage disequilibrium, LD) between the pair of SNPs. Three values of LD have been calculated and may be selected: D', r^2, and LOD (default); darker shades indicate higher LD values. White squares indicate pairwise D' values less than 1 with no statistically significant evidence of LD. Blue squares indicate pairwise D' values of 1 but without statistical significance (LOD < 2). 
sub encodeLod($$)
{
    my $lod    = shift @_;
    my $dPrime = shift @_;
    $ret = ord('a');
    if ($lod>=2) # high LOD
        {
        if (abs($dPrime)<0.5) { $ret = ord('y'); } # high LOD, low D'  -> pink
        else { $ret += min((int($lod-abs($dPrime)-1.5)), 9) ;}        
	}
    elsif (abs($dPrime)>0.99) { $ret = ord('z'); } # high D', low LOD  -> blue
    return chr($ret);
}

Methods

Display Conventions and Configuration

  • Display Mode
    • Full mode shows the pairwise LD values in a HaploView-style mountain plot.
    • Dense mode shows the pairwise LD values in a single line for each population, where the intensity at each position is the average of all of the LD values for all SNPs within 250 kb.
  • LD Values
    • r^2
    • D'
    • LOD
  • Track Geometry
    • Trim to triangle shows the standard mountain plot (default); turning this option off will show LD values with SNPs outside the window.
    • Inverting makes it easier to visually compare two adjacent populations.
  • Colors
    • LD Values can be drawn in a variety of colors.
    • Outlines can be drawn in contrasting colors or turned off.
  • Population Selection
    The four HapMap populations can be displayed or hidden individually.
