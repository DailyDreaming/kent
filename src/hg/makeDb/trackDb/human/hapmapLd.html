Description

This track shows three different measures (D', r^2, and LOD) of linkage disequilibrium (LD) between pairs of SNPs as genotyped by the HapMap consortium. Linkage disequilibrium is a measure of the association between two markers. This is useful to understand which regions of the genome are commonly inherited in blocks without recombination. See the Recombination Rates and Recombination Hotspots tracks for related analyses.

In full mode, each diagonal represents a different SNP, with each square representing a pairwise comparison between two SNPs. Shades of red are used to indicate linkage disequilibrium between the pair of SNPs, with darker shades indicating stronger LD.

Methods

Genotypes from HapMap Phase II release 19 were used with HaploView to infer phasing and calculate LD values for all SNP pairs within 250kb.

Display Conventions and Configuration

  • Display Mode
    • Full mode shows the pairwise LD values in a HaploView-style mountain plot.
    • Dense mode shows the pairwise LD values in a single line for each population, where the intensity at each position is the average of all of the LD values for all SNPs within 250 kb.
  • LD Values: measures of linkage disequilibrium
    • r^2
      This displays the raw r^2 value for a given marker pair. SNPs that have not been separated by recombination, r^2 = 1; in this case, these two markers are said to be redundant. Lower r^2 values show a lower degree of LD, indicating that some recombination has occurred in this population.
      One useful property of r^2 is that its inverse, 1/r^2, provides a practical estimate of the magnitude by which the sample size must be increased in a study design to detect association between the phenotype and a marker locus, as compared to the size required for detecting association with the susceptibility locus itself. Please see Pritchard and Prezowrski (2001) for details.
    • D'
      This displays the raw D' value for a given marker pair. A D' value of 1 (complete LD) indicates that two SNPs have not been separated by recombination, while lower values indicate that the ancestral allele has undergone some recombination during the history of the population. Only D' values near 1 are a reliable measure of LD; lower values are difficult to interpret as the magnitude of D' depends stongly on sample size. Please see Lewontin (1988) for more details.
      White squares indicate pairwise D' values less than 1 with no statistically significant evidence of LD. Blue squares indicate pairwise D' values of 1 but without statistical significance (LOD < 2).
    • LOD
      This displays the log odds score for linkage disequilibrium between a given marker pair.
  • Track Geometry
    • Trim to triangle shows the standard mountain plot (default); turning this option off will show LD values with SNPs outside the window.
    • Inverting makes it easier to visually compare two adjacent populations.
  • Colors
    • LD Values can be drawn in a variety of colors, with red as default. The intensity of the color is proportional to the strength of the LD measure chosen above.
    • Outlines can be drawn in contrasting colors or turned off.
  • Population Selection
    The four HapMap populations can be displayed or hidden individually.
