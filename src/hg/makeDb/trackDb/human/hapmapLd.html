Description

This track shows three different measures (D', r2, and LOD) of linkage disequilibrium (LD) between pairs of SNPs as genotyped by the HapMap consortium. Linkage disequilibrium is a measure of the association between two markers. This is useful to understand which regions of the genome are commonly inherited in blocks. See the Recombination Rates and Recombination Hotspots tracks for related analyses.

By default, LOD values are displayed in full mode. Each diagonal represents a different SNP with each square representing a pairwise comparison between two SNPs. Shades are used to indicate linkage disequilibrium between the pair of SNPs, with darker shades indicating stronger LD. White squares indicate pairwise D' values less than 1 with no statistically significant evidence of LD (LOD < 2). Blue squares indicate pairwise D' values of 1 but without statistical significance.

Methods

Genotypes from HapMap Phase II release 19 were used with HaploView to infer phasing and calculate LD values for all SNP pairs within 250kb.

Display Conventions and Configuration

  • Display Mode
    • Full mode shows the pairwise LD values in a HaploView-style mountain plot.
    • Dense mode shows the pairwise LD values in a single line for each population, where the intensity at each position is the average of all of the LD values for all SNPs within 250 kb.
  • LD Values: measures of linkage disequilibrium
    • r2
      This displays the raw r2 value for a given marker pair. SNPs that have not been separated by recombination have r2 = 1; in this case, these two markers are said to be redundant. Lower r2 values show a lower degree of LD, indicating that some recombination has occurred in this population. Please see Pritchard and Przeowrski (2001) for details.
    • D'
      This displays the raw D' value for a given marker pair. A D' value of 1 (complete LD) indicates that two SNPs have not been separated by recombination, while lower values indicate that the ancestral allele has undergone some recombination during the history of the population. Only D' values near 1 are a reliable measure of LD; lower values are difficult to interpret as the magnitude of D' depends strongly on sample size. Please see Lewontin (1988) for more details.
    • LOD
      This displays the log odds score for linkage disequilibrium between a given marker pair, and is shown by default.
  • Track Geometry
    • Trim to triangle shows the standard mountain plot (default); turning this option off will show LD values with SNPs outside the window.
    • Inverting makes it easier to visually compare two adjacent populations.
  • Colors
    • LD Values can be drawn in a variety of colors, with red as default. The intensity of the color is proportional to the strength of the LD measure chosen above.
    • Outlines can be drawn in contrasting colors or turned off.
  • Population Selection
    The four HapMap populations can be displayed or hidden individually.
