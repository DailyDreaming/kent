The April 2005 release of the Purple Sea Urchin genome (Strongylocentrotus purpuratus) was produced by Baylor College of Medicine's Human Genome Sequencing Center, and corresponds to their Spur_0.5 whole genome shotgun assembly.

Sample position queries

A genome position can be specified by a linkage group coordinate range, the accession number of an mRNA or EST, a gene name, or keywords from the GenBank description of an mRNA. The following list shows examples of valid position queries for the S. purpuratus genome. Note that some position queries (e.g. "brachyury") may return matches to the mRNA records of other species. In these cases, the mRNAs are mapped to their homologs in S. purpuratus. See the User's Guide for more information.

Assembly Details

Excluding gaps, the size of the preliminary assembled genome is 835 Mb, and is comprised of 187,943 linearized scaffolds. The N50 size of the scaffolds is 55 Kb. The Atlas assembler was used.

More assembly details are in the Spur_0.5 README file, and on the Baylor HGSC Sea Urchin Genome Project Web Page.
