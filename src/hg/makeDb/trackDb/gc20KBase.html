Description

The GC percent track shows the percentage of G (guanine) and C (cytosine) bases in a 20,000 base window. High GC content is typically associated with gene-rich areas.

Using the Filter

The track filter can be used to configure display attributes of the track's graph.

  • Type of graph: The default "Bar" setting depicts the graph data using filled-in bars. To switch to a view in which the data are represented by points or lines, select the "Points" setting.
  • y = 0.0 line: Select the "ON" setting to display a line marking the 0.0 position on the graph (default is "OFF").
  • Track height: To change the default display height of the graph in pixels, type in a value between 11 and 128 (the default display height is 36 pixels).
  • Vertical viewing range: By default, the viewing range is set to the values of min: 30, max: 70. Change these values to move this view window over the data. Can be used as a threshold indicator, for example, enter a minimum value of 50 to view GC percent values only above 50%.
  • Data view scaling: The default "Use Vertical Viewing Range Setting" displays the data using the parameters specified in the Vertical viewing range setting. To configure the graph to automatically scale to a range defined by the minimum and maximum data points in the current view, select the "Auto-Scale to data view" option.
  • Windowing function: At views that are too large to show individual data values, the data values must be combined to produce a plot point. This option selects the combining function. Maximim will select the maximum value of all the points to be combined, Mean selects the mean, Minimum selects the minimum of all the points to be combined.
  • Smoothing window: Sets the size, in pixels, of a smoothing window to be passed over the plot. Can be used to smooth the edges of the bar plot. This is equivalent to a trend line calculation on the graph.
  • Draw indicator line at y=: Draw an arbitrary line on the graph at the selected value. Can be used to indicate a significant threshold on the graph.

When you have finished configuring the filter, click the Submit button.
