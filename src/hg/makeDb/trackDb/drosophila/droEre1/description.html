Sample position queries

A genome position can be specified by a contig coordinate range, the accession number of an mRNA or RefSeq entry, a gene name, or keywords from the GenBank description of an mRNA. The following list shows examples of valid position queries for the D. erecta genome. Note that some position queries (e.g. "huntington") may return matches to the mRNA records of other species. In these cases, the mRNAs are mapped to their homologs in D. erecta. See the User's Guide for more information.

Use this last format for author queries. Although GenBank requires the search format Zeng LW, internally it uses the format Zeng,L.W.

Assembly details

Agencourt produced the 28 October 2004 assembly using the Arachne assembler. The assembly contains 18,915 contigs ranging in size from 335 bases to 196,364 bases, with a mean size of mean 7061.9 and median of 1698.

Bulk downloads of the sequence and annotation data are available via the Genome Browser FTP server or the Downloads page. The droEre1 annotation tracks were generated by UCSC and collaborators worldwide. See the Credits page for a detailed list of the organizations and individuals who contributed to the success of this release.
