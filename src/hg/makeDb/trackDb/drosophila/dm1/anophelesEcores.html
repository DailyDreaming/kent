Description

This track shows Evolutionary Conserved Regions computed by the Exofish program at Genoscope.

Methods

See Assessing the Drosophila melanogaster and Anopheles gambiae genome annotations using genome-wide sequence comparisons. Jaillon et al. Genome Res. 2003 Jul;13(7):1595-9. The sequence versions used in the comparison were Ensembl Drosophila v.16.3a.1 (Jan. 2003, the same as BDGP Release 3.1 used by the UCSC Genome Browser) and Ensembl Anopheles v.16.2.1 .
