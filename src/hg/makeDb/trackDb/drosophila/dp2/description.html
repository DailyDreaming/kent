The August 2003 Drosophila pseudoobscura genome assembly is based on the Freeze 1 assembly produced at the Baylor Human Genome Sequencing Center (HGSC).

Sample position queries

A genome position can be specified by the accession number of a sequenced genomic clone, an mRNA or EST or STS marker, a scaffold coordinate range, or keywords from the GenBank description of an mRNA. The following list shows examples of valid position queries for the D. pseudoobscura genome. See the User's Guide for more information.

Request:
    Genome Browser Response:
Contig3128_Contig3153 Displays all of scaffold Contig3128_Contig3153
Contig53_Contig54:1-50,000 Displays first 50,000 bases of scaffold Contig53_Contig54

AI501130 Displays region of EST with GenBank accession AI501130
AF199335 Displays region of mRNA with GenBank accession AF199335
apoe Displays region of genome with gene identifier apoE
NM_145881 Displays region of genome with RefSeq identifier NM_145881

pseudogene mRNA Lists transcribed pseudogenes but not cDNAs
zinc finger Lists many zinc finger mRNAs
kruppel zinc finger Lists only kruppel-like zinc fingers
huntington Lists candidate genes associated with Huntington's disease
Jones,R. Lists mRNAs deposited by co-author R. Jones

Use this last format for author queries. Although GenBank requires the search format Jones R, internally it uses the format Jones,R..

Assembly details

The Freeze 1 assembly is a whole-genome shotgun assembly, using Baylor HGSC's assembly engine Atlas, with approximately 7x coverage of the euchromatic portion of the genome. There are 759 scaffolds in the assembly. Scaffold N50 is 1,018,646 bp; total size is 139.3 Mbp; average 184,465 bp. Four large scaffolds were split (due to size) into "A" and "B" parts in the downloadable assembly files:
  • Contig3286_Contig7811 was split into Contig3286_Contig7811A and Contig3286_Contig7811B.
  • Contig4971_Contig7717 was split into Contig4971_Contig7717A and Contig4971_Contig7717B.
  • Contig6811_Contig7852 was split into Contig6811_Contig7852A and Contig6811_Contig7852B.
  • Contig7891_Contig7492 was split into Contig7891_Contig7492A and Contig7891_Contig7492B.
Baylor HGSC has made available a putative chromosome assignment for the majority of larger scaffolds (> 90% of unique sequence), based on conservation between the Muller elements.

Bulk downloads of the sequence and annotation data are available via the Genome Browser FTP server or the Downloads page. These data are made available with specific conditions for use (see also here). The dp2 annotation tracks were generated by UCSC and collaborators worldwide.
