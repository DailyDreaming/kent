Description

This track contains the alignments of the JGI predicted transcripts to the $Organism genome.

Methods

JGI's predicted peptides were aligned to their predicted transcripts using the BLAT program to establish the coding portion of the transcript. The transcripts were then aligned to the $Organism genome using BLAT.
To read more information about this assembly and an analysis of the euchromatic regions of the genome, see Dehal et al., The Draft Genome of Ciona intestinalis: Insights into Chordate and Vertebrate Origins. Science. 2002 Dec 13;298(5601):2157-67.
