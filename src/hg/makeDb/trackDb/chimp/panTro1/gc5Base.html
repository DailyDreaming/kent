Description

This track indicates GC counts in a sliding window of 5 bases.

Methods

To accentuate the difference between AT and GC rich regions, the scale of the graph runs from -2 to +3 for the range of GC counts from 0 to 5. Thus, GC counts of 3, 4 and 5 are graphed as positive values 1,2 and 3 in blue. GC counts of 0 and 1 are graphed as negative values -2 and -1 in red. A GC count of 2, which is the mean, appears as a red line at value 0.

Suggested improvement: GC count of 2 can not be seen easily at the zero value.

Using the Filter

The track filter can be used to configure display attributes of the track's graph.

  • Type of graph: The default "Bar" setting depicts the graph data using filled-in bars. To switch to a view in which the data are represented by points or lines, select the "Points" setting.
  • Show y=-0.0 line: Select the "ON" setting to display a line marking the 0.0 position on the graph (default is "OFF").
  • Track height: To change the default display height of the graph in pixels, type in a different value between 11 and 128 (the default display height is 36 pixels).
  • Vertical viewing range: By default, the viewing range is set to the values of min: -2, max: 3. Type in a different value to reduce the range of data values displayed.
  • Data view scaling: The default "Use Vertical Viewing Range Setting" displays the data using the parameters specified in the Vertical viewing range setting. To configure the graph to automatically scale to a range defined by the minimum and maximum data points in the current view, select the "Auto-Scale to data view" option.

When you have finished configuring the filter, click the Submit button.
