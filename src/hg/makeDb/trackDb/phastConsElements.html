Description

This track shows predictions of conserved elements from the phastCons program. PhastCons bases its predictions on a phylogenetic hidden Markov model (phylo-HMM), a type of probabilistic model that describes both the process of DNA substitution at each site in a genome and the way this process changes from one site to the next.

Methods

Best-in-genome pairwise alignments were generated for each species using blastz, followed by chaining and netting. A multiple alignment was then constructed from these pairwise alignments using multiz. A phylogenetic model was fitted to the multiz alignments by maximum likelihood, conditioning on the accepted tree topology. This was accomplished with the program phyloFit, using the general reversible (REV) substitution model and discrete gamma model for rate variation (Yang, 1994). (With some sets of species, special allowances are made for distant species, which can be aligned only in conserved regions. Branch lengths to these species are estimated approximately, using branch length proportions inferred from sites in 3rd codon position.)

The multiz alignments were split into fragments of approximately 1Mb in size, such that cut-points occurred between alignment blocks where possible. The phastCons program was run on each of these fragments with the --viterbi option and the globally estimated phylogenetic model. PhastCons constructs a two-state phylo-HMM with a state for conserved regions and a state for non-conserved regions. It "cuts" the estimated distribution of evolutionary rates at a designated quantile and creates a phylogenetic model for the conserved state based on the slower evolutionary rates and a phylogenetic model for the non-conserved state based on the faster evolutionary rates. When run with the --viterbi option, the program identifies segments of the alignment that are likely to have been "generated" by the conserved state--i.e., maximal segments in which the maximum-likelihood (Viterbi) path through the phylo-HMM completely resides in the conserved state. These predicted segments are combined across alignment fragments and used to construct the track. The transition probabilities between the states in the phylo-HMM are approximately estimated by maximum likelihood.
