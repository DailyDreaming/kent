Sample position queries

A genome position can be specified by the accession number of an mRNA or EST, a chromosome range, or keywords from the GenBank description of an mRNA. The following list provides examples of various types of position queries for the Fugu genome. Note that some position queries (e.g. "huntington") may return matches to the mRNA records of other species. In these cases, the mRNAs are mapped to their homologs in Fugu. See the User Guide for more help.

Use this last format for entry authors -- even though GenBank searches require Evans JE format, GenBank entries themselves use Evans,J.E. internally.

Assembly details

This assembly was constructed with using Whole Genome Shotgun (WGS) resulting in a sequence coverage of about 7.9X. WGC libraries were obtained from Genoscope. A plasmid library was prepared by random shearing and size selction of DNA resulting in an average insert size of 4 kb. Genoscope has also generated reads that are from plasmid or BAC end sequences. This assembly contains 49,509 contigs totaling 319 million base pairs. Coverage is estimated to be 83% in current contigs.

Downloads of the Tetraodon data and annotations may be obtained from the UCSC FTP site or Downloads page.
