Description

A valid pair of BAC end sequences must be at least 50Kb but no more than 600Kb away from each other. The orientation of the first BAC end sequence must be "+" and the orientation of the second BAC end sequence must be "-".

Methods

BAC end sequences are placed on the assembled sequence using Jim Kent's NCBI Clone Registry. You can view the entry for this clone in the registry by clicking on the clone name at the top of the page.
