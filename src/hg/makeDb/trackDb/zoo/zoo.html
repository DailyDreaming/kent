Description

This track displays the level of conservation for each of the zoo species based on blastz pair-wise alignments.

Methods

Blastz pair-wise alignments were performed between the NISC-generated sequence for each species and the UCSC compiled human reference sequence. All sequences were RepeatMasked, with repetitive sequences denoted by lower-case letters. The blastz program does not initiate alignments within repetitive regions, but extends alignments into these regions.

Gray horizontal bars denote a mapping/sequencing gap. Percent identities of less than 50% are not shown because they fall below the minimum y-axis value.
