Description

This track displays the level of conservation for each of 12 zoo species based on blastz pair-wise alignments.

Methods

Blastz pair-wise alignments were performed between the NISC-generated sequence for each species and the UCSC compiled human reference sequence. All sequences were RepeatMasked, with repetitive sequences denoted by lower-case letters. The blastz program will not initiate alignments within repetitive regions, but will extend alignments into these regions.

Gray horizontal bars denote a mapping/sequencing gap. A percent identity less than 50% is not shown because it falls below the minimum y-axis value.
