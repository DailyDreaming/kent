Description

The VAX004 DNA Sequence Conservation track shows conservation based the MSA (Multiple Sequence Alignment) result of VAX004 HIV-1 gp120 nucleotide sequences.

Methods

Conservation

At each genomic position of the MSA, the counts of different bases (A, T, C, G, "-") were tallied and the frequency distribution calculated. The frequency (fraction) of the most frequent base at each genomic position is used as the conservation measure. The result is compiled into a wig format file and loaded to construct this track.

MSA method

We applied two local (L-INS-i and E-INS-i) and one global (G-INS-i) iterative refinement algorithms. MAFFT also incorporates a BLOSUM62 scoring matrix (an empirical matrix calculated from large datasets representing extreme protein family diversity) and assumes a fixed frequency for each AA for building MSAs; however, the estimated relative rates of changes in this matrix may be too general to fit datasets of specific gene families or high substitution rates, such as gp120. Hence, here we performed multiple MAFFT alignments using both the BLOSUM62 and gp120-specific scorings matrixes and/or fixed and variable AA frequencies estimated for the data at hand. Gene-specific empirical matrices and AA frequencies based on the VAX004 and LAGB-VAX004 datasets were generated via the program MATRIXGEN (http://matrixgen.sourceforge.net).

The quality of the MSA was then compared using the maximum likelihood (ML) tree building approach in PhyML (Guindon and Gascuel 2003) under the GTR+G+I model. ML trees showing the highest likelihood scores were indicative of better alignments. G-INS-i + BLOSUM62 + AA specific frequencies generated the trees with the best likelihood scores.

Uncertainty in our MSAs was assessed using GBlocks v0.91b (Castresana 2000). Ambiguous blocks were selected and removed according to the following settings: minimum number of sequences for a conserved position = 517, minimum number of sequences for a flank position = 877, maximum number of contiguous non-conserved positions = 8, minimum length of a block = 10, and allowed gap positions = all. These settings generated a conserved MSA of 1,401 sites.
