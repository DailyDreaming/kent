Description

This track displays tRNA genes predicted by using tRNAscan-SE v.1.23.

tRNAscan-SE is an integrated program that uses tRNAscan [1] and an A/B box motif detection algorithm [2] as pre-filters to obtain an initial list of tRNA candidates. The program then filters these candidates with a covariance model-based search program COVE [3] to obtain a highly specific set of primary sequence and secondary structure predictions that represent 99-100% of true tRNAs with a false positive rate of fewer than 1 per 15 gigabases.

Detailed tRNA annotations for eukaryotes, bacteria, and archaea are available at Genomic tRNA Database (GtRNAdb).

Both tRNAscan-SE and GtRNAdb are maintained by the Lowe Lab at UCSC.
