Assembly details

This whole genome shotgun assembly was constructed with the Baylor assembler, Atlas, achieving a coverage of 3X. It contains 470,000 scaffolds.

For more information about the bosTau1 assembly, see the Baylor Bovine project page.

Bulk downloads of the sequence and annotation data are available via the Genome Browser FTP server or the Downloads page. The bosTau1 annotation tracks were generated by UCSC and collaborators worldwide. See the Credits page for a detailed list of the organizations and individuals who contributed to the success of this release.
