Sample position queries

A genome position can be specified by the accession number of an mRNA or EST, a scaffold range, an Ensembl transcript ID, or keywords from the Genbank description of an mRNA. The following list provides examples of various types of position queries for the zebrafish genome. Note that some position queries (e.g. "huntington") may return matches to the mRNA records of other species. In these cases, the mRNAs are mapped to their homologs in zebrafish. See the User Guide for more help.

Assembly details

This assembly was constructed using two different methods. First, a whole genome shotgun (WGS) approach was employed. DNA from 1000 five-day-old Tuebingen embryos was used to generate plasmid and fosmid libraries. The libraries were sequenced and the resulting traces were assembled with the Sanger Centre assembler, Phusion. The second approach used traditional clone mapping and sequencing to produce a higher quality genome sequence. BAC libraries were fingerprinted after HindIII restriction enzyme digestion and the fingerprinted contigs (FPC) program were used to build contigs from fingerprinted clones and to generate an FPC map. After generation of a tiling path that covers the genome sequence from this map, clones were selected from the path for high quality sequencing and then pieced together to form the genome sequence.

The Zv3 assembly comprises 1,459,115,486 bp in 58,339 supercontigs resulting in a sequence coverage of about 5.7X. This zebrafish assembly is the first to be tied to the FPC map: 1,083,447,588 bp (74%) of the sequence were mapped in this way.

Note: The Sanger Institute warns that this assembly is still preliminary; there is a high level of misassembly due to polymorphisms in the DNA source. Highly variable regions of the assembly are difficult to assemble, most likely due to differing haplotypes.

Downloads of the Zebrafish data and annotations may be obtained from the UCSC FTP site or Downloads page. For more information about this assembly see the Sanger Centre page for the Danio rerio Sequencing Project.
