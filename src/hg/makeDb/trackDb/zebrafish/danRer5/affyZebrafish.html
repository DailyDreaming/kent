Description

This track shows the location of the consensus sequences and the target sequences used for the selection of probes on the Affymetrix Zebrafish genome chip.

Display Conventions and Configuration

This track follows the display conventions for composite tracks. This track may be configured to vary the type of sequence aligned - either consensus or target sequences or both.

Methods

Consensus and target sequences were downloaded from the Affymetrix Product Support web page. The sequences were mapped to the genome with blat followed by pslCDnaFilter using the parameters -localNearBest=0.005, -minQSize=20, -minNonRepSize=16, -ignoreNs, -bestOverlap, -minId=0.95 and -minCover=0.15.
