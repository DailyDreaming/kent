Description

This track shows the location of the consensus sequences used for the selection of probes on the Affymetrix Zebrafish genome chip.

Methods

Consensus sequences were downloaded from the Affymetrix Product Support web page. The sequences were mapped to the genome using blat followed by pslReps with the parameters -minAli=0.95 and -nearTop=0.005.
