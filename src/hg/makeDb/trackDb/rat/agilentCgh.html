Description

This track displays the probes from the Agilent Catalog Oligonucleotide CGH Microarrays.

Agilent's oligonucleotide aCGH platform is an integrated solution that facilitates performing genome-wide DNA copy number variation profiling without amplification or complexity reduction. Probe coverage spans both coding and non-coding regions, with emphasis on well-characterized genes.

The track consists of the following subtracks:

Rat Genome CGH Microarray 1x244K AMADID 015223

This annotation displays the probe positions for the Rat Genome CGH 1 x 244K microarray from Agilent Technologies, Inc.
  • 235,000+ coding and non-coding mouse sequences represented
  • 7.9 kb overall median probe spatial resolution (4.7 kb median probe spacing in Refseq genes)
  • 1 x 244K slide format

Rat Genome CGH Microarray 2x105K AMADID 015235

This annotation displays the probe positions for the Rat Genome CGH 2 x 105K microarray from Agilent Technologies, Inc.
  • 97,000+ coding and non-coding mouse sequences represented
  • 19.1 kb overall median probe spatial resolution (11.3 kb median probe spacing in Refseq genes)
  • 2 x 105K slide format
