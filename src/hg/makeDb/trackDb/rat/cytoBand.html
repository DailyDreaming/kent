Description

The chromosome band track represents the approximate location of bands seen on Giemsa-stained chromosomes at an 400 band resolution.

Methods

Data is derived from ideogram file downloaded from the NCBI ftp site ftp://ftp.ncbi.nih.gov/genomes/R_norvegicus/maps/mapview/. Band lengths are estimated based on ISCN defined relative sizes.
