Description

This track shows locations of STSs (Sequence Tagged Sites) along the draft assembly. These STSs have been mapped using either genetic (rat FHH x ACI F2 intercross genetic map, rat SHRSP x BN F2 intercross genetic map) or radiation hybridization (RH map.2.2).

Additional data on the individual maps can be found at the following links:
  • NCBI UISTS
  • RAT GENOME DATABASE
