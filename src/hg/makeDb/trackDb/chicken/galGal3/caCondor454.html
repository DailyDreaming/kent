Description

This track displays the alignment of California Condor (Gymnogyps californianus) transcripts sequenced using 454 high-throughput DNA sequencing.

Methods

  1. 454 read sequences were obtained the NCBI Trace Archive (CENTER_NAME = WUGSC, CENTER_PROJECT = U_FC-MATT_TECHD2). Mean read size is 110 bases.
  2. Align condor reads to a set of predicted transcripts sequences and project the read to transcript alignments to the genome using the predicted transcript alignments. This process improves alignments across splice sites.
    • build a set of predicted transcripts sequences using PASA cluster of Chicken mRNA and EST BLAT alignments and N-SCAN gene predictions.
    • align condor reads to transcripts using LASTZ
    • project alignments to the genome
    • filter alignments to remove duplicate alignments and small fragments, keep those that score withing one 1% of the best alignment for a given read.
  3. Align reads that could not be mapped via the predicted transcript directly to the genome using LASTZ, and filter using the above criteria.
