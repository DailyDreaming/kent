Description

Reference alignment of mouse genome vs. the human genome constructed from the Mouse Translated Blat Alignments track.

Methods

The track is constructed by selecting the best-scoring blat alignment for each aligned position in the genome. Mouse insertions are included in the reference alignment when they follow the last base of a blat block, although this process is not entirely perfected.

Credits

This track is produced at UCSC. Mouse sequence data is provided by the Mouse Genome Sequencing Consortium.
