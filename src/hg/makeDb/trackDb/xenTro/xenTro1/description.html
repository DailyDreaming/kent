The October 2004 frog (Xenopus tropicalis) whole genome shotgun (WGS) assembly release version 3.0 was sequenced and assembled by the DOE Joint Genome Institute (JGI).

Sample position queries

A genome position can be specified by the accession number of a sequenced genomic clone, an mRNA or EST, a chromosomal coordinate range, or keywords from the GenBank description of an mRNA. The following list provides examples of various types of position queries for the frog genome. See the User's Guide for more information.

Request:
    Genome Browser Response:

scaffold_3 Displays all of scaffold 3
scaffold_3:1-2000000 Displays first 2 million bases of scaffold_3
AY572227 Displays location of mRNA with GenBank accession AY572227
BM538765 Displays location of EST with GenBank accession BM538765
scaffold_1001.8 Displays location of Genscan gene prediction with identifier scaffold_1001.8

pseudogene mRNA Lists transcribed pseudogenes but not cDNAs
zinc finger Lists zinc finger mRNAs
breast cancer Lists mRNAs associated with breast cancer susceptibility proteins
evans Lists mRNAs deposited by scientist named Evans
Murphy,K.E. Lists mRNAs deposited by co-author K.E. Murphy

Use this last format for author queries. Although GenBank requires the search format Murphy KE, internally it uses the format Murphy,K.E..

Assembly Details

The assembly release version 3.0 of whole genome shotgun reads was constructed with the JGI assembler, Jazz, using paired end sequencing reads at a coverage of 7.4X. After trimming for vector and quality, 19.1 Million reads assembled into 27,064 scaffolds totaling 1.63 Gbp. Roughly half of the genome is contained in 392 scaffolds all at least 1.2 Mb in length. For more information about this assembly, see the JGI Xenopus tropicalis site.

The xenTro1 sequence and annotation data can be downloaded from the UCSC Genome Browser FTP server or downloads page. These data have specific conditions for use.
