Description

This shows that best blastp match to the known genes protein set from the UCSC mouse genome browser database.

Clicking

Clicking here will open UCSC Mouse Genome Browser window on the gene.

Method

NCBI blastall was run with the following parameters:
-p blastp -e 0.005 -m 8 -b 1

Thanks to the Mouse Sequencing Consortium, the Mammalian Gene Collection, and the Riken Fantom project for the data.
