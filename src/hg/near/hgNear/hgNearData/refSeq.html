Description

This column displays the NCBI RefSeq accession associated with each gene, if available. RefSeq genes are a non-redundant set of high-quality mRNA sequences.

Action

Click on the accession number to display the NCBI LocusLink record for a RefSeq accession. If LocusLink shows a link to the Online Mendelian Inheritance in Man (OMIM) -- represented by an orange square with an "O" inside it -- an OMIM record is available for the gene. OMIM is often an excellent source of human-curated information about human genes.
