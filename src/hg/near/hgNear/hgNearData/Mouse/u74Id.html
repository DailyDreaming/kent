Description

This track the Affymetrix ID from the U74 series of chips (A, B, and C) that corresponds best to this gene

Methods

Target genome sequences were downloaded from the Affymetrix Download Center and mapped to the genome using blat followed by pslReps with the parameters -minCover=0.85 -minAli=0.95. For each gene the most overlapping target (on the same strand and excluding introns) is chosen.
