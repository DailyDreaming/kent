Description

This shows that best blastp match to the known genes protein set from the UCSC human genome browser database.

Clicking

Clicking here will open UCSC Human Genome Browser window on the gene.

Method

NCBI blastall was run with the following parameters:
-p blastp -e 0.001 -m 8 -b 1

Thanks to the International Human Sequencing Consortium, the Mammalian Gene Collection, and GenBank contributors worldwide for the data.
