Description

This column shows the Affymetrix ID from the HG-U95 chip that best corresponds best to each gene.

Methods

Target genome sequences were downloaded from the Affymetrix Download Center and mapped to the genome using Blat followed by PslReps with the parameters -minCover=0.85 -minAli=0.95. For each gene, the most overlapping target (on the same strand and excluding introns) was chosen.
