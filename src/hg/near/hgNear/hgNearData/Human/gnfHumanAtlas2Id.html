Description

This column shows the ID of the probe in the GNF Atlas 2 which overlaps most with gene from the from the Human Gene Expression. The GNF Altas 2 is based on two Affymetrix chips, the U133A, and a custom-designed GNF1H chip.

Methods

Target sequences for U133A were downloaded from the Affymetrix Download Center. Target sequences for GNF1H were obtained from GNF via email. The sequences and mapped to the genome using blat followed by pslReps with the parameters -minCover=0.85 -minAli=0.95. For each gene, the most overlapping target (on the same strand and excluding introns) was chosen.
