Description

This column shows data from the GNF Expression Atlas on the Affymetrix U95 chip. It shows the expression of genes in specific tissues. This track can be configured a number of ways.

The track initially shows the ratio of expression in a specific tissue to expression of the gene overall. By default if a gene is more highly expressed in a tissue it will be colored red, and if it is less expressed in the tissue it will be green. This coloring is standard, but the opposite of what a naive user might think. In this case red means go, and green means stop! Color blind users can switch from red/green to yellow/blue coloring in the configuration page. The track can also be configured to show absolute expression values as shades of gray with higher expression levels shown in lighter shades. Both the ratio and absolute expressions are colored on a logarithmic scale.

The 'tissue' drop-down in the configuration page controls the level of detail displayed. If 'all replicas' is selected then the value of each individual experimental relica of each tissue is shown. If median of replicas is selected then just a single value for each tissue is shown. The default 'selected' mode just shows median values for selected tissues.

Methods

For detailed information about the experiments, see Su et al., "Large-scale analysis of the human and mouse transcriptomes.", PNAS, Mar 19, 2002.

When calculating expression ratios, the overall expression level in the denominator is calculated by first taking the median of replicants for each non-cancerous tissue, and then taking the median of these medians.
