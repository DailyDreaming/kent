Description

This column displays all the Protein Data Bank (PDB) ID's associated with a gene. PDB is a database of proteins with known 3-D structures. In some cases these records will only correspond to only a fragment of the gene. In other cases the PDB record may include other molecules the protein interacts with.

Action

Click on the PDB ID to display the associated PDB Structure Explorer page.

Methods

The PDB ID's are extracted from SwissProt.
